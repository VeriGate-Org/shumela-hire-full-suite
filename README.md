# ShumelaHire

A comprehensive recruitment and hiring management platform built with Next.js 15, React 19, TypeScript, and Tailwind CSS.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Frontend**: Next.js 15 + React 19 + TypeScript + Tailwind CSS 3.4
- **Backend**: Spring Boot (Java) — `backend/` directory
- **Font**: Manrope (via `next/font/google`)
- **Design**: Arthmatic DevWorks brand guidelines

## Development

```bash
# Install dependencies
npm install

# Run frontend dev server
npm run dev

# Run backend
cd backend && mvn spring-boot:run

# Run tests
npm test

# Build for production
npm run build
```
